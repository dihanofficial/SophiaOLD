### Self-hosting (For Devs) ‚öî
```sh
# Install Git First (apt-instll git)
$ git clone https://github.com/DihanOfficial/Sophiabot
$ cd SophiaBot
# Upgrade sources
$ bash deploy.sh
# Install All Requirements 
$ pip3 install -r requirements.txt
# Rename ./SophiaBot/data/bot_conf.yaml.example to bot_conf.yaml and fill
# Start Bot 
$ python3 -m SophiaBot
```
